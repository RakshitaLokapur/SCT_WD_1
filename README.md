The objective of this task is to design and develop a responsive landing page with an interactive navigation menu. # Responsive Landing Page with Interactive Navigation

This project implements a modern and responsive landing page with an interactive navigation menu that enhances user experience through smooth hover effects and scroll-based style changes.

Task Overview

The goal of this task is to create a responsive landing page featuring a fixed navigation bar that:
- Remains visible on all sections
- Changes its appearance when the page is scrolled
- Highlights menu items on hover
- Adapts smoothly to different screen sizes

The project focuses on clean UI design, usability, and frontend interactivity using core web technologies.

## Features

- Fixed navigation bar visible across all sections
- Interactive hover effects on navigation links
- Navbar style changes dynamically on scroll
- Modern UI/UX with gradients and glassmorphism effects
- Smooth transitions and animations
- Fully responsive for desktop, tablet, and mobile devices

## Technologies Used

- **HTML5** – Page structure and layout  
- **CSS3** – Styling, animations, and responsiveness  
- **JavaScript** – Scroll-based interactivity
